Project 1:
Technology Trend Analysis

March 2024

Project Goal: Discover what programming languages, database skills, and web frames are the most in demand using data from job postings and surveys.

Introduction
This case study resembles the work a Junior Data Analyst would do for a global IT and business consulting services firm that is known for its expertise in IT solutions. To keep pace with changing technologies and remain competitive, this organization regularly analyzes data to help identify future skill requirements. I determined these future skill requirments using data from the 2019 Stackoverflow developer survey, web scraping, and web APIs. This analysis is split into six sections: Ask, Prepare, Process, Analyze, Vizualize, and Act. The scripts I created for this analysis can be found here: Jupyter Notebook files .


Ask
What programming languages, database skills, and Web Frames are the most popular now and are likely to be popular to the future? Properly understanding technology trends is vital in the digital age because it has historically given companies a competetive advantage.

Prepare
I used Beautiful Soup web scraping to find average annual salary by technology, and a web API to find number of jobs by location and technology. After importing this data to a pandas dataframe, I used Openpyxl to convert the dataframes to Excel files. These Excel files were used to create the Number of Jobs by City, Number of Jobs by Technology, and Average Annual Salary by Technology bar graphs in the share section. Finally I created a data frame from the 2019 Stackoverflow developer survey data, which will be used in the analyze section.

Process
I started cleaning the Stackoverflow data by deleting duplicate rows using the drop_duplicates function. I found missing values in the "Work Location" column using np.nan and replaced them with the most frequent value in the column. Finally, the compensation column was inconsistant because it contained weekly, monthly, and annual compensation. I fixed this problem by creating a "normalized annual compensation" column that multiplyed weekly compensation by 50 (assuming two weeks of holidays per year), monthly compensation by 12, and kept annual compensation the same.

Analyze
After cleaning the data, I used histograms and box plots to understand the distribution of annual salary and age for survey respondents.


Share
I started making visualizations by creating bar charts from the Excel files created with Openpyxl.


After generating the three bar graphs in Excel, I created this dashboard in Cognos Analytics with three tabs: Current Technology Usage, Future Technology Trend, and Survey Demographics.

Act
The top four findings based on my analysis:

  1. JavaScript is the leading programming language, but Python is close to surpasing JavaScript. TypeScript is also fast growing, and could potentially convert JavaScript users.
  2. MySQL is the most popular database, and PostSQL and MongoDB are gaining popularity. Microsoft SQL Server and SQLite should be avoided due to decreasing popularity.
  3. JQuery, React.js, and Angular/Angular.js are the most popular webframes, but JQuery's growth seems to be stagnating.
  4. Linux and Windows are the most popular platforms, and Docker is quickly approaching them in popularity.
